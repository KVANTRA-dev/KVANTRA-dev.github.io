## Что делает NOUZ

<div class="lead-panel">
  <p>NOUZ превращает папку Markdown-заметок в управляемый граф знаний. Вы задаёте смысловую рамку и принимаете решения, а сервер считает домены, проверяет связи, собирает состав узлов и отдаёт агенту структурированный контекст через MCP-инструменты.</p>
</div>

## Инструменты как философия

NOUZ не пытается заменить мышление автоматикой. Его инструменты делают структуру видимой: один инструмент читает заметку, другой показывает родителей и детей, третий предлагает разметку, четвёртый пересчитывает доменный состав базы. Агент получает не “кучу файлов”, а набор ясных операций над графом знаний.

Это главный принцип проекта: знания остаются локальными и человечески управляемыми, а вычисления помогают увидеть связи, дрейф и пробелы в структуре.

## Иерархия

Граф строится сверху вниз. Каждый уровень отвечает за свою роль, а для больших баз можно добавить `meta_root` — якорный узел уровня 0, который собирает домены в одну систему и не участвует в семантических расчётах.

| Уровень | Тип | Что сюда помещать | Откуда знак |
| ------- | --- | ----------------- | ----------- |
| L1 | Ядро | Домен знаний — широкая область | Вручную или по эталонам |
| L2 | Паттерн | Тематическая рамка для дочерних модулей и квантов | Вручную или по эталонам |
| L3 | Модуль | Функциональная группа, проект | Вручную или по эталонам |
| L4 | Квант | Одна идея — гайд, исследование, пост | Тип артефактов детей + домен по контенту |
| L5 | Артефакт | Сырьё: логи, чаты, конфиги | Тип по эвристике содержимого |

## Быстро попробовать

```bash
pip install nouz-mcp
```

Создайте `config.yaml` из шаблона или используйте режим LUCA по умолчанию:

```yaml
mode: prizma
```

Укажите путь к папке с заметками и подключите эмбеддинги (для PRIZMA/SLOI):

```bash
export OBSIDIAN_ROOT=/path/to/vault
export EMBED_API_URL=http://127.0.0.1:1234/v1
```

Запустите через MCP-клиент:

```json
{
  "mcpServers": {
    "nouz": {
      "command": "python",
      "args": ["-m", "nouz_mcp"],
      "env": {
        "OBSIDIAN_ROOT": "/path/to/vault"
      }
    }
  }
}
```

Режим **LUCA** работает без эмбеддингов — только YAML и граф. [→ Быстрый старт](/nouz/quick-start)

## Теоретический контекст

NOUZ — практический MCP-сервер, его можно использовать без теоретической части. Для тех, кому интересна исследовательская рамка проекта: [«Рекурсивная самоорганизация как универсальный принцип»](https://doi.org/10.5281/zenodo.19595850).
